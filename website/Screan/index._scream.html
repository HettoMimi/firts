CSS son las siglas en inglés de Cascading Style Sheets, que significa “hojas de esilo en cascada”. Es un lenguaje que se usa para estilizar elementos escritos en un lenguaje de marcado como HTML.CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996 por una razón muy sencilla.
